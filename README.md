# Madhhav-Sales-PowerBI-Dashboard
"Interactive Powe BI dashboard analyzing e-commerce sales performance, and customer payment behaviour to drive stategic business decisions."

## 🚀 Project Overview
This project involves the development of an interactive Business Intelligence dashboard using **Power BI** to visualize and analyze e-commerce sales data. The objective was to transform raw transactional data into actionable insights regarding profitability, category trends, and payment preferences.

## 📊 Key Features
- **Profitability Analysis:** Visualized monthly profit trends to identify peak performance periods.
- **Category Insights:** Analyzed 'Sum of Quantity' by product category to understand consumer demand.
- **Payment Behavior:** Evaluated payment mode distribution (COD, UPI, Credit/Debit cards) to understand customer transaction preferences.
- **Sub-Category Breakdown:** Detailed view of profit distribution across various product sub-categories.

## 🛠️ Tools & Technologies Used
- **Software:** Microsoft Power BI
- **Data Analysis:** DAX, Data Modeling
- **Version Control:** Git & GitHub

## 📈 Insights Gained
- Identified key contributors to total profit and volume.
- Pinpointed seasonal trends and fluctuations in sales performance.
- Evaluated the efficacy of various payment methods in the sales cycle.
